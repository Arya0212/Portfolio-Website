Real-time Yoga Pose Detection and Classification

Developed a robust yoga pose classifier utilizing custom data and Mediapipe for keypoint extraction. Integrated CNN, LSTM with attention mechanism for enhanced precision. Deployed using Tkinter for intuitive user interface. Leveraged TensorFlow, Keras, scikit-learn, among other frameworks for development and optimization.

Web-Based Mass Mail Dispatcher

The Mass Mail Dispatcher is an innovative web application designed to simplify and enhance the process of sending emails to multiple recipients simultaneously. Leveraging modern web technologies and the EmailJS API, the project aims to provide users with an efficient, user-friendly, and customizable solution for mass email dispatch.
